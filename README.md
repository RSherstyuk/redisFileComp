# File Compressor

A multithreaded file compressor using hiredis and zlib.

## Requirements
- C++17
- CMake 3.10+
- Zlib
- Hiredis

## Build
```bash
mkdir build && cd build
cmake ..
cmake --build .
```

Usage:

## Producer
./consumer <outpur_dir>

## Consumer
./producer <input_dir>
